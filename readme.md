# weather-visualizer
## 目录
```
weather-visualizer/
├── app.py               # 项目主入口，启动网页应用
├── requirements.txt     # 依赖列表，pip install -r requirements.txt
├── readme.md            
├── data/                # 存放数据
├── model/               # 训练好的模型文件
├── static/              # CSS
└── templates/           # HTML 
```


## 运行步骤

1. 安装依赖：

```bash
pip install -r requirements.txt
````

2. 启动程序：

```bash
python app.py
```

3. 打开浏览器访问：

```
http://127.0.0.1:5000
```
